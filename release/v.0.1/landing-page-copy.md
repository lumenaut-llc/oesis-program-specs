## Intro section

Open Environmental Sensing and Inference System is part of Open Source DIY Tech.
The project is designed so a single home can benefit on its own, while
neighborhood participation can improve precision and confidence over time. The
system is built around a simple rule set: private by default, shared by choice,
explicit about uncertainty, and intended for a broader commons-protective open
release.

This open release makes the current governance, data ownership, privacy posture,
IP/release boundaries, approved technical materials, and high-level project direction
public under asset-specific terms.

## What makes it different

### Parcel first

The project aims to generate parcel-level condition estimates instead of relying only on coarse regional signals.

### Homeowner controlled

Parcel-linked data remains homeowner-controlled rather than automatically becoming
public or platform-owned. The project-controlled v1 dataset may be intentionally
public when explicitly designated and licensed, but future participant data is not
made public by default.

### Modular and standalone

The system is designed so modular builds can work independently and still interoperate as part of a broader network.

### Honest about uncertainty

Outputs are meant to carry confidence, freshness, and evidence-mode information rather than pretending every result is equally certain or directly observed.

## What this release includes

- the project mission, architecture, and current release posture
- the current governance and privacy posture
- the data ownership and sharing model
- the release and IP position for the current release period
- the intentionally public v1 dataset carve-out and dataset policy
- claims and limitations language for responsible public communication

## What this release does not include

- every file in the repository as a public release asset
- secrets, non-cleared third-party materials, or unintended personal / operational data
- any blanket open-data release of future participant-contributed parcel-linked data
- claims that the project replaces official alerts, emergency services, or on-scene judgment
