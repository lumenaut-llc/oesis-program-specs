# v0.3 Gap Register

## Purpose

Classify gaps between the current implementation state and v0.3 readiness.

## Gap classification key

- **Blocker** — must resolve before claiming v0.3
- **Resolved** — gap closed by implementation
- **Defer** — acceptable to track as known gap; schedule for later slice
- **Carry-forward** — inherited from earlier gap registers, not v0.3 scope
