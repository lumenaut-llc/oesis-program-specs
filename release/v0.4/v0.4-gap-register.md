# v0.4 Gap Register

## Purpose

Classify gaps between the current implementation state and v0.4 readiness.

## Gap classification key

- **Blocker** — must resolve before claiming v0.4
- **Defer** — acceptable to track; schedule for later slice
- **Carry-forward** — inherited from earlier registers, not v0.4 scope

## Gap register

| ID | Gap | Severity | Owner area | Current status | Next action |
