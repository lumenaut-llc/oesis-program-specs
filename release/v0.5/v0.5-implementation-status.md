# v0.5 Implementation Status

## Purpose

Track per-surface implementation status for the v0.5 release.

## Status key

- `implemented` — code exists, acceptance test passes
- `partial` — code or contract exists but acceptance test not yet passing
- `docs-only` — specification exists, no runtime implementation
- `planned` — in scope but no specification or implementation
- `n/a` — not in v0.5 scope

### Data class governance detail

The runtime enforces structural privacy rules for 9 data classes:

| Class | Governance | Reason |
|-------|-----------|--------|
| indoor_pm25 | Structurally private | Occupancy pattern inference |
| indoor_temperature | Structurally private | Occupancy pattern inference |
| hvac_mode | Structurally private | Equipment vulnerability |
| sump_pump_state | Structurally private | Infrastructure vulnerability |
| power_state | Structurally private | When home is unprotected |
| action_logs | Structurally private | Household behavior |
| outcome_records | Structurally private | Household behavior |
| io_ratio_baseline | Structurally private | Building envelope fingerprint |
| thermal_profile | Structurally private | Building envelope fingerprint |

Shareable classes (outdoor_pm25, outdoor_temperature, outdoor_humidity,
outdoor_pressure, local_flood_stage, parcel_state_label, divergence_signal)
require active consent and are subject to minimum participation thresholds
before aggregation.

## Acceptance test coverage note

The v0.5 acceptance test (`make oesis-v05-accept`) validates:

- Structurally private data class rejection on consent grant (must fail)
- Valid consent grant with temporal + spatial resolution
- Governance status query returns granted consent
- Governance private summary lists private classes
- Consent revocation with revoked_at timestamp
- History query shows revocation lifecycle
- Shared map output suppresses revoked parcel contributions
- Retention cleanup removes old entries, preserves recent
- Export bundle produces output with metadata

It does **not** test HTTP-level governance enforcement, concurrent access,
or consent re-grant after revocation.
