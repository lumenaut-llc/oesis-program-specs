# OESIS Public Site Pointer

This directory no longer hosts the active public preview site application.

The canonical public site repository now lives at:

- `../oesis-public-site`

Use the sibling site repo for:

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

From this specs repository, `make public-site-build` and related site targets
proxy to `../oesis-public-site`.
