# Version labels and lanes

This file is the short glossary for version language in this repository and in the numbered operating packet (`01`–`11`). Use it to avoid mixing **program phases**, **runtime asset lanes**, and **public release** naming.

**Canonical incorporation:** The same vocabulary is summarized in [`program/v0.1/README.md`](../README.md) (program phase labels), in [`architecture/system/version-and-promotion-matrix.md`](../../architecture/system/version-and-promotion-matrix.md) (axes and promotion table), and connected to milestones in [`architecture/current/milestone-roadmap.md`](../../architecture/current/milestone-roadmap.md). Pre-1.0 slice ladder: [`architecture/current/pre-1.0-version-progression.md`](../../architecture/current/pre-1.0-version-progression.md).

## Program phase `v0.2` (formal next promotion)

**`v0.2` is not deprecated** in program-specs. It is the formal name for the **next accepted runnable slice** after `v0.1` in [`architecture/current/pre-1.0-version-progression.md`](../../architecture/current/pre-1.0-version-progression.md): stable **indoor + sheltered outdoor** operation (`bench-air-node` + `mast-lite`) with explicit architecture, contract, acceptance, and implementation-status evidence.

The sibling **`oesis-runtime`** repository may or may not expose a separate asset directory or Makefile target named `v0.2`. That does not change the **program-specs** promotion label: follow the runtime `README.md` for which commands exercise widened-kit behavior until the runtime aligns naming.

## This glossary vs numbered packet files

Numbered files in this folder (`01`–`11`, etc.) are program framing and phasing briefs. **Formal architecture, contracts, and the promotion matrix** are canonical under [`architecture/`](../../architecture/) (especially [`architecture/system/version-and-promotion-matrix.md`](../../architecture/system/version-and-promotion-matrix.md)). If older packet text disagrees with `pre-1.0-version-progression.md` or the matrix, **prefer the architecture docs**.
