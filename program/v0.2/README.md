# Program v0.2

`v0.2` is a promotion lane marker for the first widened parcel kit.

## Program posture

- Inherit baseline program docs from `../v0.1/`.
- Add files here only if a `v0.2`-specific program delta is explicitly accepted.
- Until such a delta exists, this directory is intentionally lightweight.

## What v0.2 means for the program

- Program scope widens from single bench-air node to two-node parcel kit
- Indoor plus sheltered outdoor operation becomes the accepted baseline
- mast-lite joins bench-air as a coordinated system, not an optional add-on

## How to use this lane

- For current work, use `../v0.1/` and `../operating-packet/`.
- Add files in this lane only when a concrete `v0.2` program delta is accepted.
- Do not copy `v0.1` files here by default.
