# Program v0.2

`v0.2` is a promotion lane marker for the first widened parcel kit.

## Sign-off sentence

**v0.2** means: bench-air + mast-lite bound to one parcel, both streams
normalized and combined, parcel view explains source mix — without claiming
field-hardened deployment, governance enforcement, or flood/thermal coverage.

## What v0.2 means for the program

- Program scope widens from single bench-air node to two-node parcel kit
- Indoor plus sheltered outdoor operation becomes the accepted baseline
- mast-lite joins bench-air as a coordinated system, not an optional add-on
- Parcel view must explain which evidence came from indoor vs outdoor nodes

## Scope boundaries

- **In scope**: Two-node kit, multi-source inference, source-attributed parcel view
- **Not in scope**: Field-hardened deployment, governance enforcement, flood/thermal
  families, trust scoring, shared-map, intervention surfaces
