# Meta v0.2

`v0.2` is a promotion lane marker for the first widened parcel kit.

## What v0.2 means for meta

- Planning and coordination for two-node kit promotion
- Milestone tracking for indoor + sheltered outdoor acceptance
- Promotion requires: architecture scope, contract/runtime boundary changes,
  acceptance commands, and implementation-status evidence

## Promotion gate

Per `pre-1.0-version-progression.md`, v0.2 promotion requires:

1. Explicit architecture scope → `release/v0.2/v0.2-scope-matrix.md`
2. Contract and runtime boundary changes → documented in acceptance criteria
3. Acceptance commands → `make oesis-v02-accept` (proposed)
4. Implementation-status evidence → `release/v0.2/v0.2-implementation-status.md`

## How to use this lane

- Inherit baseline meta docs from `../v0.1/`.
- Add files here only if a `v0.2`-specific planning delta is explicitly accepted.
