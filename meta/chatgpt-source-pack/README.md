# ChatGPT Source Pack

This folder replaces the older dated path-forward source-pack markdowns with a smaller canonical set organized by present state, next promotion, longer-range direction, and executable system shape.

Each file starts with a short synthesis and then pastes the full content of the source files it summarizes so the pack can be dropped directly into a ChatGPT project without losing the underlying context.

## Path convention

When release references use `release/v.0.1/`, read that as release label
`v0.1` with filesystem path `v.0.1/`.

## Files

- `01-current-state.md` — current truthful state across specs and runtime.
- `02-next-step.md` — the next honest promotion and what must change to earn it.
- `03-direction.md` — future-lane and v1.5 bridge direction, clearly separated from current truth.
- `04-system-shape.md` — executable and contract-level app shape from the inference engine and runtime APIs.
