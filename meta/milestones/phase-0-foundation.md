# Phase 0 — Foundation

## Goal

Create the shared project frame so every workstream can operate with aligned context.

## Deliverables

- canonical docs uploaded to the project workspace
- repo scaffold created
- subsystem stubs created
- glossary and naming stabilized
- first procurement bundles defined
- privacy, claims, licensing, and pilot-governance foundation documented
- machine-readable governance-side data contracts defined
- release and pilot checklists drafted

## Exit criteria

- hardware and software workstreams can proceed independently
- legal and governance workstreams have stable inputs
- documentation structure is usable
- core governance decisions are captured in ADRs
- pilot planning can proceed without inventing privacy and claims posture from scratch
