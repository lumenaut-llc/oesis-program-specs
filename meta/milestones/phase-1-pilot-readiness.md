# Phase 1 — Pilot Readiness

## Goal

Translate the documented governance model into actual product behavior, operator workflows, and pilot controls.

## Deliverables

- sharing settings implemented in product and backend flows
- consent records and rights requests implemented from schemas
- parcel platform shows provenance, freshness, sharing state, and rights controls
- shared-map suppression, aggregation, and disclosure rules implemented
- operator access logging and pilot incident path operational
- participant-facing pilot notice and consent flow integrated

## Exit criteria

- a pilot can run in private-only mode by default with optional sharing clearly separated
- no public parcel-by-parcel map is exposed for real contributed data
- revocation stops future sharing in practice, not only in policy
- export and deletion requests have a real intake and tracking path
- public and pilot copy use condition-estimate language consistently
