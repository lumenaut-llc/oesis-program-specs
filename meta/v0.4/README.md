# Meta v0.4

`v0.4` is a promotion lane marker for multi-node registry and evidence composition.

## What v0.4 means for meta

- Planning and coordination for registry lifecycle and evidence composition
- Milestone tracking for multi-node management acceptance
- Requires v0.3 promotion as prerequisite baseline

## Promotion gate

1. Explicit architecture scope → `release/v0.4/v0.4-scope-matrix.md`
2. Contract and runtime boundary changes → lifecycle states, deployment_grade, weighting
3. Acceptance commands → `make oesis-v04-accept` (proposed)
4. Implementation-status evidence → `release/v0.4/v0.4-implementation-status.md`

## How to use this lane

- Inherit baseline meta docs from `../v0.1/`.
- Add files here only if a `v0.4`-specific planning delta is explicitly accepted.
