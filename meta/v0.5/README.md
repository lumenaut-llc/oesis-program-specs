# Meta v0.5

`v0.5` is a promotion lane marker for operational governance enforcement.

## What v0.5 means for meta

- Planning and coordination for governance enforcement promotion
- Milestone tracking for consent, revocation, retention, and export acceptance
- Requires v0.4 promotion as prerequisite baseline
- v0.5 is a prerequisite for v1.0 — governance must be enforced before broader system

## Promotion gate

1. Explicit architecture scope → `release/v0.5/v0.5-scope-matrix.md`
2. Contract and runtime boundary changes → consent-store, sharing-store become operational
3. Acceptance commands → `make oesis-v05-accept` (proposed)
4. Implementation-status evidence → `release/v0.5/v0.5-implementation-status.md`

## How to use this lane

- Inherit baseline meta docs from `../v0.1/`.
- Add files here only if a `v0.5`-specific planning delta is explicitly accepted.
