# Preview Site Controls

## Purpose

Keep the release-owned publication controls and guidance for the public preview
site after the Astro app moved to a stable top-level location.

## What lives here

- `public-content-allowlist.md`
- `public-preview-guide.md`
- this README

## App location

The active Astro app now lives in:

- `sites/public-preview/`

## What this folder now governs

- which routes and anchors are approved for the public preview
- which source materials may feed the public site
- which materials must stay out of public navigation
- how the site should be reviewed against release boundaries

## Review rule

The public site is intentionally high-level and non-enabling. Before publishing
it externally, review it against:

- `../open-source-v1-summary.md`
- `../../../../legal/public-preview-scope.md`
- `../../../privacy-governance/claims-and-safety-language.md`
- `../launch-readiness-checklist.md`

Use `public-content-allowlist.md` as the release-owned publication gate for the
site app in `sites/public-preview/`.
