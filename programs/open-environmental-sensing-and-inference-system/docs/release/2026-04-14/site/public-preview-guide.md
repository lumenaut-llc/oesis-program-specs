# Public Preview Guide

## Purpose

Provide a public-facing reading guide for the preview site without turning the website into a mirror of the more technical release and governance materials.

This page is intentionally high-level and non-enabling.

## What this preview is

The April 14, 2026 preview is a public introduction to:

- the project mission and principles
- the private-by-default and shared-by-choice posture
- homeowner data ownership expectations
- the current release and governance boundary
- the project's open-release direction at a high level

It is not the full enabling technical release.

## What this preview is not

This site does not try to publish:

- implementation-enabling software details
- held-back hardware design internals
- detailed scoring, threshold, or inference logic
- real homeowner-contributed parcel-linked data
- internal reviewer or counsel workflow materials

## Website boundary

The public website should:

- summarize the project in accessible language
- point to the canonical public policy and release documents
- avoid acting like a raw repository browser
- avoid collapsing public site copy and technical release artifacts into one document layer

## Related internal release docs

The broader release packet also includes more technical or reviewer-oriented materials such as the asset-class matrix and contributor review guide.

Those docs may inform the website, but the website should usually pull from them selectively rather than present them as first-stop public reading.
