## Purpose

This directory now points to the versioned technical architecture canon.

For `v0.1`, the key architectural clarification is:

- parcel is the primary decision object
- collection and home/platform ingest are the primary evidence-availability
  realities
- shared intelligence is architecturally anticipated, but not the primary
  meaning of network in the first functioning version

- `../technical-architecture/README.md` for the versioned architecture entrypoint
- `../technical-architecture/v0.1/README.md` for the current truthful reference
  architecture

  functioning first-version slice
- `../technical-architecture/v0.1/architecture-object-map.md` for the current
  first-class architecture objects
- `../technical-architecture/v1.0/README.md` for the debated target architecture

The older `architecture/` path is kept temporarily so existing links do not
break during the transition.
