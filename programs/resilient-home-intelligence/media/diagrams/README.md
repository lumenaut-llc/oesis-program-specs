# Diagrams

Use this folder for diagrams.

## Current diagram packs

- `prototype-integration-diagram-pack.md`
  Public-safe diagram pack for breaking the system into parts, showing how those parts combine into one prototype, comparing architecture avenues, and exploring design possibilities.

## Usage note

Keep public-safe and controlled-review diagrams separate.
High-level architecture, topology, roadmap, and privacy-boundary diagrams may live here.
Detailed method flowcharts, reproducible implementation diagrams, and filing-sensitive figures should stay in controlled-review locations until the release posture is clear.
