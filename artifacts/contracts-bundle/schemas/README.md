# Contract Schemas

JSON Schema files compiled into the contracts bundle. Categories include:

- **Core observation:** node-observation, parcel-state
- **Governance:** consent-record, sharing-settings, sharing-store, rights-request
- **Support objects:** node-registry, shared-neighborhood-signal, operator-access-event
- **Retention:** export-bundle, retention-cleanup-report

These files are generated by the build system from source schemas in
`contracts/`. Do not edit directly.
