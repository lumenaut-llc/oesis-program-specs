# Legal v0.4

`v0.4` is a promotion lane marker for multi-node registry and evidence composition.

## What v0.4 means for legal

- Installation metadata capture introduces privacy considerations for
  site-specific data (mount location, orientation, exposure context)
- Deployment metadata privacy posture should be reviewed before field deployment
- No governance enforcement changes (deferred to v0.5)

## Legal review items

- Privacy classification of installation metadata fields
  (see `../../contracts/v1.0/deployment-metadata-schema.md`)
- Retention posture for deployment metadata records
- Operator access scope for installation metadata

## How to use this lane

- Inherit baseline legal docs from `../v0.1/`.
- Add files here only if a `v0.4`-specific legal delta is explicitly accepted.
