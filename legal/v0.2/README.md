# Legal v0.2

`v0.2` is a promotion lane marker for the first widened parcel kit.

## What v0.2 means for legal

No major legal changes over `v0.1`. The two-node kit operates under the same
licensing and governance posture. No sharing API is exposed; private-by-default
is enforced architecturally.

## How to use this lane

- Inherit baseline legal docs from `../v0.1/`.
- Add files here only if a `v0.2`-specific legal delta is explicitly accepted.
