# Legal v0.3

`v0.3` is a promotion lane marker for the first flood-capable runtime.

## What v0.3 means for legal

No major legal changes. Flood-node operates under the same licensing and
governance posture as existing node families. Private-by-default remains
architectural; no sharing API exposed.

## How to use this lane

- Inherit baseline legal docs from `../v0.1/`.
- Add files here only if a `v0.3`-specific legal delta is explicitly accepted.
