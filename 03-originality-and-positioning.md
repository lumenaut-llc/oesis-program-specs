# Originality and positioning

**Canonical incorporation:** Strategic framing overlaps [`architecture/system/vision-and-use-cases.md`](architecture/system/vision-and-use-cases.md) and [`program/README.md`](program/README.md). This file keeps positioning detail.

## What this project is not most original for

This project is not most original as a sensor invention.
It is not strongest as a claim that nobody else has built environmental sensing networks before.

## What it is original for

It is most original as a synthesis.

Its distinctiveness comes from combining:
- parcel-first reasoning
- partial-adoption operation
- homeowner-owned raw data
- opt-in shared intelligence
- multi-hazard scope
- explicit evidence modes and uncertainty
- route, block, and lifeline expansion over time
- a path from sensing toward adaptation and verified response

## Best originality claim

The right originality claim is:

original in system design, governance, inference framing, and multi-scale integration

not:

original because it invented environmental sensing from scratch

## Positioning sentence

This project is not original as a sensor invention.
It is original as a systems architecture and governance model that connects parcel sensing, uncertainty-aware inference, partial-adoption shared intelligence, and long-term resilience learning.

## Positioning caution

Do not claim:
- that the project invented hyperlocal sensing
- that no one has built community sensor systems
- that the project replaces official systems

Do claim:
- parcel-first reasoning
- multi-scale resilience intelligence
- privacy-preserving shared evidence
- owner-controlled raw data
- a staged bridge from sensing to functional resilience
